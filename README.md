# Supermarket Sales Analysis and Prediction System

A web-based system for analyzing sales data and predicting product demand using machine learning.

## Tech Stack
- **Backend**: Django (Python)
- **Frontend**: React
- **Database**: PostgreSQL
- **ML**: Python (scikit-learn/TensorFlow)

## Project Structure

```
sales-analysis/
├── backend/          # Django backend
│   ├── apps/        # Django apps
│   └── ml_module/    # ML prediction module
├── frontend/         # React frontend
└── docs/             # Documentation
```

## Features
- ✅ User authentication with role-based access (Admin, Sales Manager, Staff)
- ✅ Multi-shop support
- ✅ Sales billing system
- ✅ Inventory/Stock management
- ✅ Inter-shop stock transfers
- ✅ Sales analytics and reports
- ✅ ML-powered demand predictions
- ✅ Automated alerts (low stock, high demand)
